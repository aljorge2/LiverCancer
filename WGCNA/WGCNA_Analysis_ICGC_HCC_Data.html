Purpose: To perform the WGCNA analysis on the ICGC Hepatocellular Carcinoma data

Methods: Following the tutorial from https://bioinformaticsworkbook.org/tutorials/wgcna.html#gsc.tab=0 to perform this analysis. Will note decisions and assumptions as I go through in order to doublecheck them with the team.

Calculate and plot WGCNA network

allowWGCNAThreads()
## Allowing multi-threading with up to 8 threads.
picked_power = 30 ## Selected as the value which maximized the R squared value
temp_cor <- cor
cor <- WGCNA::cor


netwk <- blockwiseModules(tumorAdjacentExpSubset,                # <= input matrix

                          # == Adjacency Function ==
                          power = picked_power,                # <= power here
                          networkType = "signed",

                          # == Tree and Block Options ==
                          deepSplit = 2,
                          pamRespectsDendro = F,
                          # detectCutHeight = 0.75,
                          #minModuleSize = 30,
                          #maxBlockSize = 4000,

                          # == Module Adjustments ==
                          reassignThreshold = 0,
                          mergeCutHeight = 0.25,

                          # == TOM == Archive the run results in TOM file (saves time)
                          saveTOMs = T,
                          saveTOMFileBase = "ER",

                          # == Output Options
                          numericLabels = T,
                          verbose = 3)
